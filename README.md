# Python高校实验室管理系统

## 核心功能
- 用户登录注册
- 实验室信息管理
- 设备信息管理
- 实验任务发布与跟踪
- 规章制度管理
- 实验室课表安排
- 通知公告发布
- 设备申请与维修管理
- 学生预约实验
- 教师预约实验

## 适用场景
- 高校实验室日常管理
- 实验室设备资源分配
- 实验室教学与科研支持
- 实验室安全与规章制度执行

## 技术特点
- 使用Django框架进行后端开发
- 利用MySQL进行数据存储
- 前端采用Vue技术实现用户界面
- 提供源码、开发文档和数据库文档
